## Codebase Understanding

### Architecture Overview

This is a layered FastAPI + sqlite3 + Python-CLI codebase. Web app under `src/web/` (Fly-deployed; web container ships `revue_core` separately as a wheel — **importing revue_core from web modules at top-level crashes uvicorn**, per REVUE-345 precedent in `src/web/jwt_signing.py:23-25`). CLI/skill under `packaging/revue/src/revue_skill/` shipped as a Nuitka-compiled wheel to PyPI. REVUE-278 adds a runtime licence-validation contract that:
1. Caches `/v2/licence/validate` server responses for 24h at `~/.config/revue/licence-cache.json`
2. Gates all review subcommands at `/revue-local` invocation start
3. Persists per-invocation telemetry via `POST /v2/usage/emit`
